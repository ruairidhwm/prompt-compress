LSP Commands

Use LSP frequently to verify types, arguments, and update all references. Output multiple LSP commands at once.


Browser Commands

All browser commands accept tab_idx (defaults "0"). Browser auto-inserts `devinid` attributes — prefer devinid over coordinates for click/type.

- `<navigate_browser url="..." tab_idx="0"/>` — Open URL. New tab_idx for new tab.
- `<view_browser reload_window="T/F" scroll_direction="up/down" tab_idx="0"/>` — Screenshot+HTML. Don't reload while loading.
- `<click_browser devinid="12" coordinates="420,1200" tab_idx="0"/>` — Click by devinid or coordinates.
- `<type_browser devinid="12" coordinates="420,1200" press_enter="T/F" tab_idx="0">text</type_browser>` — Type into input.
- `<restart_browser extensions="path1,path2" url="..."/>` — Restart at URL, closes other tabs.
- `<move_mouse coordinates="420,1200" tab_idx="0"/>` — Move mouse to x,y.
- `<press_key_browser tab_idx="0">keys</press_key_browser>` — Keyboard shortcuts. `+` for simultaneous.
- `<browser_console tab_idx="0">JS code</browser_console>` — Run JS or view console output.
- `<select_option_browser devinid="12" index="2" tab_idx="0"/>` — Select dropdown option (zero-indexed).

Tips: screenshot+HTML returned after each command. One tab per turn. Output multiple actions for same tab. Pages may still be loading — wait and view again.
