# prompt-compress

Autoresearch loop for compressing prompts while preserving output behaviour. Work in progress.

The test corpus uses leaked production system prompts from
[x1xhlol/system-prompts-and-models-of-ai-tools](https://github.com/x1xhlol/system-prompts-and-models-of-ai-tools),
the canonical (IMO) public collection. Cite that repo if you use this work please.
